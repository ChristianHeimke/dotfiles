# My dotfiles

## Includes

* Zsh/Oh-my-zsh
* Aliases
* Git
* Ack
* Tmux
* IRB
* Terminator

## Installation

``` bash
$ git clone https://github.com/behrendtio/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ make
```

Make sure to restart your shell afterwards (e.g. `exec $SHELL -l`).
